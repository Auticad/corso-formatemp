# Python Data Lab

Raccolta di esercizi, notebook e script SQL sviluppati durante un percorso
di formazione pratica su Python e analisi dati.

## Contenuto

| Cartella / File | Descrizione |
|---|---|
| `esercizi/` | Esercizi Python su strutture dati, funzioni e logica |
| `gestione_file_txt/` | Script per lettura, scrittura e manipolazione di file di testo |
| `Progetto_github/sql-script/` | Script T-SQL per creazione e interrogazione di database |
| `main.py` | Esempi di sintassi Python avanzata (walrus operator, list comprehension) |
